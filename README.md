# LumaGrid

LumaGrid is a modern React starter built for products that need a clean foundation, strong UI/UX, and fast iteration. It is designed to be easy to develop, easy to extend, and visually polished from the beginning.

## Overview

This project gives you a lightweight frontend structure using React, Vite, and TypeScript. The current version ships with a responsive landing page, reusable content sections, and a visual system that feels premium without adding unnecessary complexity.

## Why LumaGrid

- Easy to understand project structure
- Fast local development with Vite
- Responsive layout for desktop and mobile
- Clean component-based setup for future expansion
- Strong visual identity with modern typography, layered backgrounds, and smooth motion

## Tech Stack

- React
- TypeScript
- Vite
- Plain CSS with a reusable design system

## Architecture

The project follows a simple frontend structure so it stays approachable as the app grows.

```text
LumaGrid/
|- index.html
|- package.json
|- vite.config.ts
|- tsconfig.json
|- src/
|  |- main.tsx
|  |- App.tsx
|  |- styles.css
```

## Working Flow

1. `index.html` loads the app root and fonts.
2. `src/main.tsx` mounts the React application.
3. `src/App.tsx` defines the content structure and reusable UI sections.
4. `src/styles.css` controls layout, responsiveness, colors, spacing, and animation.
5. Vite serves the app locally and builds it for production.

## Input and Output

### Input

- Developer updates text, sections, cards, and links in the React components
- Developer adds new screens or features inside `src/`
- Browser viewport and user interaction drive the responsive UI behavior

### Output

- A polished landing page with strong visual hierarchy
- Responsive experience across mobile, tablet, and desktop
- Production-ready static frontend build generated by Vite

## UI/UX Highlights

- Hero-first layout with clear call-to-actions
- Intentional typography using expressive display and clean reading fonts
- Glass-style cards and soft gradients for visual depth
- Comfortable spacing and readable content rhythm
- Mobile-friendly behavior without needing a separate code path

## Local Development

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

## Production Build

Create an optimized production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Future Improvements

- Add routing for multiple pages
- Convert sections into reusable component files
- Connect a backend or API layer
- Add forms, authentication, or dashboard screens
- Introduce theme switching if the product needs it
